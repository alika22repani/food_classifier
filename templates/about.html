Tentang Project

NusantaraVision

NusantaraVision adalah aplikasi web untuk mengklasifikasikan jenis makanan Indonesia dari sebuah foto, dibangun menggunakan pendekatan Transfer Learning dengan arsitektur VGG16 yang telah dilatih sebelumnya (pre-trained) pada dataset ImageNet, kemudian di-fine-tune secara ringan pada dataset makanan Indonesia.

Arsitektur Model
  • Base model: VGG16 (ImageNet weights, include_top=False)
  • Global Average Pooling
  • Dense(256, ReLU) + Dropout(0.5)
  • Dense(128, ReLU) + Dropout(0.3)
  • Dense(N kelas, Softmax)
  • Fine-tuning ringan pada blok konvolusi ke-5
Dataset
  • Sumber: Kaggle — yourboys/indonesian-food
  • Diunduh otomatis via kagglehub
  • Subset ± 3.000–5.000 gambar (dipilih proporsional per kelas)
  • Split training/validasi: 80% / 20%
  • Resize ke 224×224 piksel + preprocess_input VGG16
Augmentasi Data
  • Rotasi ±20°
  • Shift horizontal/vertikal 15%
  • Shear & zoom ringan
  • Flip horizontal
  • Variasi brightness
Konfigurasi Training
  • Maksimal 10 epoch (2 tahap: head training + fine-tuning)
  • EarlyStopping (monitor val_accuracy)
  • ModelCheckpoint (simpan model terbaik)
  • ReduceLROnPlateau
  • Optimizer Adam, loss categorical_crossentropy
Tech Stack
Python Flask TensorFlow / Keras VGG16 Bootstrap 5 Kagglehub Railway
Project ini dibuat untuk tujuan pembelajaran/edukasi mengenai penerapan Transfer Learning pada kasus klasifikasi citra dan integrasinya ke dalam aplikasi web berbasis Flask.
